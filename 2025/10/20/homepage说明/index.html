基本信息

项目名称：Homepage

仓库地址：https://github.com/gethomepage/homepage

开源协议：GPL-3.0

开发语言：TypeScript / React

项目简介

Homepage 是一个开源的、自托管的仪表盘（Dashboard）项目，主要用于搭建个人或团队的 起始页 / 服务导航页。
它提供了现代化的界面和高度自定义的配置方式，可以集中展示：

🖥 常用服务链接（带图标、状态显示）

📊 系统监控（CPU、内存、硬盘、网络）

📦 Docker / Kubernetes 容器状态

⏰ 日历、天气、新闻等小组件

🔑 支持多用户与身份验证（OAuth / Authelia / Keycloak 等）

快速部署示例（Docker Compose）

访问 http://localhost:3000 即可看到仪表盘界面。
